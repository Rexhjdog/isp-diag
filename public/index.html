About isp-diag v2.0
Agent-powered internet diagnostics tool.
Each diagnostic category is handled by a specialized AI agent.
Agents use tools to run real network tests, then provide intelligent analysis.
Speed tests and WebRTC checks run locally in your browser.
Agents:
IP & Location - Identifies your IP, ISP, and geolocation
DNS Analysis - Tests DNSSEC, DoH, DNS leaks, and ECS
Security - Checks TLS, HTTPS, and security headers
Network - Tests IPv6, HTTP/3, and connectivity
Performance - Measures DNS and HTTP latency

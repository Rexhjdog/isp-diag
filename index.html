About isp-diag
Comprehensive internet diagnostics tool with Mullvad VPN detection.
All tests run locally in your browser.
No personal data is collected or stored.
What we check:
Privacy score - Overall privacy health of your connection
VPN status - Detect VPN connections (Mullvad and others)
Blacklist - Check if your IP appears on known blacklists
IP addresses - Your public IPv4 and IPv6 addresses with geolocation
ISP - Your Internet Service Provider, ASN, and location
DNS resolvers - Which DNS servers handle your queries
DNS leak test - Whether DNS queries are routed correctly
ECS - Whether your DNS leaks your IP subnet
DNS security - DNSSEC validation, DoH, and DoT support
Connection security - HTTPS, TLS version, and HSTS
WebRTC - Check for IP address leaks via WebRTC (IPv4 + IPv6)
Browser privacy - Canvas fingerprint, WebGL, timezone, language leaks
Network capabilities - IPv6, HTTP/3, ECH, connection info
Port connectivity - Test reachability of common ports
Latency - Round-trip time measurements with jitter
Speed test - Download and upload bandwidth
VPN detection powered by Mullvad
